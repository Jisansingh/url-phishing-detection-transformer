## Dataset Information

The datasets used in this project are derived from real-world sources and are not included in this repository due to size and licensing constraints.

### Phishing URLs
- Source: OpenPhish, Phishing.Database
- Description: URLs associated with confirmed phishing websites

### Legitimate URLs
- Source: Tranco top-ranked domain list
- Description: Popular and trusted legitimate domains

### Dataset Preparation
- Duplicate URLs removed
- Balanced dataset (50% phishing, 50% legitimate)
- Character-level tokenization applied
- Train, validation, and test splits created

Researchers and reviewers can reproduce the dataset by sourcing data from the above providers and following the preprocessing scripts in this repository.
